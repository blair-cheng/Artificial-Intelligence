# Artificial-Intelligence

Set as a private repository per Northwestern professor's policy. Please reach out for access (current students excluded).

**Language:** Python  
**Assignments:** 6 · **Lines:** ~4,500

The classical AI foundations. Covers the full arc from search and planning through symbolic reasoning, probabilistic inference, and neural networks — the ideas from Russell & Norvig's *AIMA* that predate deep learning but still underpin how intelligent systems decide and reason.

**Highlights:**
- Knowledge base with forward-chaining inference, rule currying, and cascading retraction with truth maintenance — the most algorithmically complex assignment, requiring three interacting subsystems (unification, inference, dependency tracking)
- Minimax with alpha-beta pruning for Checkers AI, with a 5-factor weighted evaluation function and 16 unit tests at depths 1–4
- A* search on real Evanston/Chicago street network data, Bayesian network exact inference by enumeration
